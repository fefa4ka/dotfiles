 | Window Management  |                                                            |
| -------------------| ---------------------------------------------------------- |
| Spaces                                                                          |
| `Alt-[1-9]`        | Select N Space                                             |
| `Ctrl-Shift-[qw]`  | Move Space back/next                                       |
| `Cmd-Shift-c`      | Create space                                               |
| `Cmd-Alt-c`        | Create space for window                                    |
| `Cmd-Shift-w`      | Destroy Space                                              |
